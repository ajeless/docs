# Ideas Folder

This folder holds speculative notes, synthesis documents, sketches, and concept explorations.

Documents here may contain:

- strong intuitions
- candidate metaphors
- possible technologies
- half-baked structures
- unresolved tensions
- ideas worth preserving before they are validated

## Important rule

Material in this folder should not be treated as settled architecture.

If an idea becomes well-supported and stable, it should graduate into the main docs. If a true commitment is made, it should be recorded in `../decisions/`.
