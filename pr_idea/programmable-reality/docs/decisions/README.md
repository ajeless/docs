# Decisions Folder

This folder is intentionally sparse.

Use it to record actual decisions once they exist. Until then, keep exploration in `../ideas/` or the main docs.

## Suggested format

Decision records should answer:

- What was decided?
- Why was it decided?
- What alternatives were considered?
- What tradeoffs were accepted?
- What assumptions or dependencies does it rely on?

## Important rule

Do not put attractive ideas here just because they feel coherent.

A decision should only be recorded when the project is genuinely willing to treat it as a commitment.
